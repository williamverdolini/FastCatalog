Multi-attribute Search Proof of Concept

The Multi-attribute Search is a very common functionality in e-commerce application.
In this PoC, I moved this functionality into NoSQL realm, using MongoDb and ElasticSearch as my landing databases.

The idea to demonstrate is that this functionality could be effectively managed with NoSQL database, specially with those that are born to be used in "aggregation-based" functionalities. And ElasticSearch is one of those.
If you want to read more about this PoC, you can find my point of view and my thoughts in these series of posts.

Use the toolbar above to view "in action" the prototyped functionality in different NoSQL databases.
